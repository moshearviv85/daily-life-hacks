# Monetization Framework

## Principle
- Authority comes before monetization.
- No offer should weaken trust or push the site toward unsafe medical positioning.

## Offer Pillars
### Recipes
- pantry staples
- cooking tools
- meal prep tools

### Nutrition
- beginner guides
- printable cheat sheets
- non-medical educational resources

### Tips
- storage containers
- kitchen organizers
- prep tools

## Placement Rules
- Do not place affiliate offers above the fold as the main purpose of the page.
- Variant CTA should first drive owned audience growth.
- Affiliate CTA should appear only after trust-building or strong intent.
- Avoid explicit medical or cure-style framing in any monetized placement.

## Offer Mapping
- `fiber-guide` -> nurture into educational recommendations
- `recipe-pack` -> nurture into tools and pantry product recommendations
- `kitchen-checklist` -> nurture into organizers, storage, and prep systems

## Review Standard
- Every affiliate placement must answer:
  - Is it useful?
  - Is it aligned with the article?
  - Would it still feel honest if there were no commission?
